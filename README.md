# 🚀 小红书智能解析工具

一个集成AI投流建议的小红书无水印媒体解析工具，不仅能解析高清内容，更能为您的内容运营提供专业的投流策略建议。

## ✨ 功能特性

### 🎯 核心功能

-   **无水印解析**：自动去除小红书水印，获取高清原图/原视频
-   **智能识别**：自动识别分享链接，粘贴即用
-   **实时预览**：解析结果即时预览，支持图片轮播和视频播放
-   **🤖 AI投流建议**：基于GLM-4模型的专业投流策略分析

### 🧠 AI投流建议功能

-   **内容定位分析**：量化话题匹配度，识别标签冗余与缺口
-   **受众画像建议**：精准地域定位、兴趣标签、人群特征分析
-   **精准投流策略**：话题扩展、人群定向、竞品借势、投放时机
-   **优化建议**：标题升级、标签重构，预估CTR提升幅度

### 🎨 用户体验

-   **响应式设计**：完美适配手机、平板、桌面设备
-   **现代化 UI**：渐变主题色，流畅动画效果
-   **操作便捷**：简洁直观的操作界面
-   **加载优化**：骨架屏加载，提升用户体验

### ⚡ 技术特性

-   **极速解析**：优化的 API 请求，快速获取媒体内容
-   **AI集成**：集成智谱AI GLM-4模型，提供专业投流建议
-   **类型安全**：完整的 TypeScript 类型定义
-   **组件化架构**：高度可复用的 Vue 组件设计
-   **错误处理**：完善的错误提示和重试机制

## 🤖 AI投流建议详细功能

### 智能分析维度

#### 1. 内容定位分析
- **话题关联性量化**：计算正文与标签的匹配度，≥80%为佳
- **核心方向提炼**：8字以内概括主卖点+情绪价值
- **话题覆盖评估**：分析在站内搜索指数和类目TOP500的覆盖率

#### 2. 受众画像建议
- **地域定位**：推荐3个一线/新一线 + 2个下沉潜力城市，附TGI指数
- **兴趣标签**：识别5个高相关圈层
- **人群特征**：提供3组精准人群包描述

#### 3. 精准投流策略
- **话题扩展**：新增3-5个搜索量5w-20w、竞争度<30的长尾话题
- **人群定向**：DMP人群包（地域+兴趣+行为）
- **竞品借势**：分析近30日涨粉≥5k的同类作者
- **投放时机**：预测未来30天内3个搜索峰值节点

#### 4. 优化建议
- **标题升级**：提供A/B两版优化方案，预估CTR提升幅度
- **标签重构**：1主标签+3长尾+1场景标签的新组合

### AI技术特色

- **专业模型**：基于智谱AI GLM-4.1V-Thinking-Flash模型
- **5年+经验**：模拟资深投流操盘手的分析思路
- **数据驱动**：结合搜索指数、竞争度、TGI等关键指标
- **实时分析**：即时生成个性化投流策略报告
- **Markdown渲染**：美观的报告展示，支持富文本格式

## 🎯 使用方法

### 基础解析功能
1. **复制分享链接**：从小红书APP中复制笔记分享链接
2. **粘贴解析**：在输入框中粘贴链接，点击解析按钮
3. **获取内容**：获取无水印高清图片或视频
4. **下载保存**：一键下载媒体文件到本地

### 🤖 AI投流建议功能
1. **解析笔记**：首先完成基础解析，获取笔记内容
2. **获取建议**：点击"一键获取"按钮，AI开始分析
3. **查看报告**：获得专业的投流策略分析报告，包括：
   - 内容定位分析与话题匹配度
   - 受众画像与地域定位建议
   - 精准投流策略与时机选择
   - 标题优化与标签重构建议

### AI分析报告示例
```
# 内容定位分析
- 话题关联性：85% 匹配度
- 核心方向：美妆教程 + 实用价值
- 话题覆盖：覆盖率78%，建议补充长尾标签

# 受众画像建议
- 地域定位：上海、深圳、杭州 + 成都、西安
- 兴趣标签：美妆护肤、时尚穿搭、生活方式
- 人群特征：18-35岁女性，中高消费能力

# 精准投流策略
- 话题扩展：#平价美妆 #学生党必备 #上班族护肤
- 投放时机：晚上8-10点，周末下午2-4点
- 竞品借势：参考@美妆达人小A 近期爆款内容
```

## 🔧 开发指南

### 添加新平台支持

1. 在 `src/components/` 创建新的解析组件
2. 在 `src/utils/` 添加对应的 API 处理文件
3. 在 `src/types/` 定义类型声明
4. 更新路由配置和标签页组件

### 样式定制

-   全局样式变量在 `src/styles/variables.css`
-   主题色可通过 CSS 变量快速修改
-   响应式断点已预设，支持移动端优先
